Firebase Setup Guide

Follow these steps to enable real-time sync between yours and Abbie's phones. Takes about 10 minutes and is completely free.

1Create a Firebase project
Go to console.firebase.google.com and sign in with a Google account.

Click "Add project", name it roadtrip-quiz, disable Analytics, then click "Create project".
2Add a web app & get your config
In your new project, click the </> (Web) icon.

Name the app roadtrip-quiz and click "Register app".

Copy the firebaseConfig object — you'll need it in step 5.
3Enable Realtime Database
In the left sidebar, go to Build → Realtime Database.

Click "Create Database", choose your region, then select "Start in test mode".

⚠️ Test mode is open — fine for personal use. If you want to lock it down, go to Rules and set .read and .write to true with an auth condition later.
4Create a GitHub repo & upload files
Go to github.com, create a free account, then click "New repository".

Name it roadtrip-quiz, make it Public, and create it.

Upload all 4 files from the ZIP: index.html, manifest.json, sw.js, and icon-192.png.

Then go to Settings → Pages, set Source to Deploy from branch → main, and click Save. Your URL will be:
https://YOUR-USERNAME.github.io/roadtrip-quiz
5Get a free Gemini API key (for quiz questions)
Go to aistudio.google.com and sign in with a Google account (the same one you used for Firebase is fine).

Click "Get API key" → "Create API key". It's genuinely free — no card needed.

Copy the key (starts with something like AIzaSy...).

⚠️ Keep this key private — never paste it into a chat message or share it publicly. Treat it like a password.
6Add both config blocks to index.html
Open index.html in a text editor.

Find REPLACE THIS WITH YOUR FIREBASE CONFIG and paste in your Firebase values from step 2.

Find YOUR_GEMINI_API_KEY_HERE and replace it with your Gemini key from step 5.

Save the file and re-upload it to GitHub, replacing the old version.
7Add to home screen on both phones
Open the GitHub Pages URL in Safari (iPhone) or Chrome (Android).

iPhone: Tap the Share button → "Add to Home Screen" → Add
Android: Tap the 3-dot menu → "Add to Home Screen" → Add

The app will appear on your home screen with its own icon and work like a native app. Both phones sync automatically via Firebase. 🎉
💡 Make your GitHub repository Private (Settings → General → Danger Zone → Change visibility) so no one else can see your API keys in the source code. GitHub Pages still works fine with private repos on free accounts tied to a GitHub Pro/Team plan — if you're on a fully free personal account and Pages requires Public, consider a simple password gate instead, or ask me to add one.
